Escribe un programa que pregunte al usuario si es culpable o no debemos usar la funciona prompt(). Asumiremos que: En caso afirmativo el usuario responderá si En caso contrario responderá no. Si el usuario responde si se escribirá por el documento «irás a la cárcel». Si el usuario responde no se escribirá por el documento web «irás a casa». En cualquier caso en el documento web se escribirá «la documentación por favor».
